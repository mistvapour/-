# 🍽️ 小李家的菜单收藏博客

一个精美的菜单收藏博客，展示每日精心准备的美食菜单。

## 📋 功能特点

- 🎨 精美的响应式设计
- 📱 移动端友好
- 🍽️ 中英文对照菜单
- 📅 按日期组织的菜单
- 🎯 简洁优雅的用户界面

## 🚀 快速开始

### 本地预览
1. 克隆此仓库
2. 在浏览器中打开 `index.html`
3. 享受浏览菜单的乐趣！

### 部署到GitHub Pages
1. Fork 此仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择主分支作为源
4. 访问 `https://mistvapour.github.io/menus`

## 📁 文件结构

```
├── index.html              # 博客主页
├── menu-2024-09-19.html    # 2024年9月19日菜单
├── menu-2024-09-25.html    # 2024年9月25日菜单
├── README.md               # 项目说明
└── 菜单收藏/               # 原始菜单文件
    ├── 菜单-2024年9月19日.html
    └── 菜单-2024年9月25日.html
```

## 🎨 自定义

### 添加新菜单
1. 创建新的HTML文件，命名格式：`menu-YYYY-MM-DD.html`
2. 在 `index.html` 中添加新的菜单卡片
3. 提交更改到Git仓库

### 修改样式
- 编辑 `index.html` 中的 `<style>` 部分
- 主要颜色变量：
  - 主色调：`#d4a373` (金色)
  - 辅助色：`#8b4513` (棕色)
  - 背景色：`#f8f6f0` (米色)

## 📝 博客文章格式

每篇菜单文章应包含：
- 标题和日期
- 菜品分类（餐前酒、主菜、配菜、甜点等）
- 中英文对照
- 菜品描述
- 特色推荐
